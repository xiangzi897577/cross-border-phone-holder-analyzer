# AGENTS.md

## 项目名称
- 中文名：跨境电商手机支架选品分析平台
- 英文名：`cross-border-phone-holder-analyzer`

## 项目定位
- 这是一个用于暑期前端实习简历投递的前后端分离项目。
- 项目聚焦手机支架这个跨境电商轻小件品类，通过 Amazon 候选商品与 1688 货源数据的对比，做利润率、物流成本、竞争强度、评分风险和推荐指数分析。
- 当前阶段仍然是 Day 1，重点是项目初始化、页面骨架和后端基础服务，不是业务功能实现。

## 当前技术事实
- 前端：`Vite + React + JavaScript`
- 后端：`Node.js + Express`
- 后端模块规范：`ES Module`
- 数据存储方案：后续使用本地 `JSON` 文件

## 当前已实现
- 已创建 `client` 前端项目，并可通过 `npm run dev` 启动。
- 已创建 `server` 后端项目，并可通过 `npm start` 启动。
- 前端当前是项目介绍型首页，不包含商品列表、筛选、图表和接口联调。
- 后端当前已实现：
  - `GET /`
  - `GET /api/health`
  - 未命中路由的 `404 JSON` 返回
